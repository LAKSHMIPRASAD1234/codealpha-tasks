📈 Stock Portfolio Tracker

📌 Description

Stock Portfolio Tracker is a simple Python application that helps users calculate the total value of their stock investments. Users enter stock names and quantities, and the program calculates the total investment based on predefined stock prices.

✨ Features

✅ User input for stock names and quantities
✅ Predefined stock prices using a dictionary
✅ Automatic investment value calculation
✅ Total portfolio value calculation
✅ Saves results to a text file
✅ Simple console-based interface

🛠️ Technologies Used

🐍 Python
📖 Dictionary
⌨️ Input/Output
➕ Basic Arithmetic
📂 File Handling

📊 Sample Stocks

📈 AAPL : $180
🚗 TSLA : $250
🔍 GOOGL : $140
💻 MSFT : $420
🛒 AMZN : $190

▶️ How to Run

1️⃣ Open the project folder.
2️⃣ Run the file:

python tracker.py

3️⃣ Enter stock names and quantities.
4️⃣ Type "done" to finish.
